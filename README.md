## Overview
This repository contains the backend of a Food Delivery Web Application.
The backend is built using Node.js, Express, and MongoDB and provides REST APIs
for authentication and core application logic.
## Backend Setup (Day 1)

- Initialized Node.js project
- Installed required packages: `express`, `mongoose`, `cookie-parser`, `dotenv`, etc.
- Setup MongoDB connection using Atlas
- Added environment variables in `.env`
- Implemented basic auth routes (`auth.route.js`)
- Added `cookie-parser` and JSON body parser in `index.js`
- Ensured `.gitignore` to exclude `node_modules` and `.env`
